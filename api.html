This API is a test api, it is still not 100% working, it may give errors, but you can use it for your discord bot or whatever you want, if you need help write to my discord PistonCube#0001


/Servers (server list)

api.pistoncube.com.ar/servers - Gives the servers where the bot is (currently not working properly)

/Players (Players list)

api.pistoncube.com.ar/players/(Server)

- gives the username,uuid, ping, displayName

Example: api.pistoncube.com.ar/players/2b2tar

will respond with something like this

[{"username":"PistonCubeBOT","uuid":"c4fd8d64-711e-3918-88e7-797149a5f363","ping":58,"displayName":{"json":{"text":"PistonCubeBOT"},"text":"PistonCubeBOT"}},

/tablist (the server tab)

api.pistoncube.com.ar/tablist/(Server)

- gives the header, footer

Example: api.pistoncube.com.ar/tablist/2b2tar

will respond with something like this

{"header":{"text":"\n§7§l2BUILDERS2TOOLS\n"},"footer":{"text":"\n§820 §8tps — 23 Players — 58 ping\n"}}
